# 🍽️ Restaurant POS & Inventory System

A full-stack Restaurant POS & Inventory Management System built using MERN Stack (React, Node.js, Express, MongoDB).

The system manages cashier operations, products, inventory tracking, orders, and reports with role-based access control.

---

# 🚀 Tech Stack

- Frontend: React.js, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB + Mongoose
- Authentication: JWT
- Image Upload: Cloudinary
- Architecture: Layered Architecture (Controller - Service - Repository)
